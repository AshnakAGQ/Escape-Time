# Escape-Time
Procedural generated first person runner game with time slowing mechanics

Game Design by Justin Kephart (justinpianist1@gmail.com)  
Spring 2018  
Moorpark College

Check the PDF for full documentation and credits

Controls  
===============================================  
•	WASD to move  
•	Spacebar to jump   
•	E to interact  
•	RMB to slow time
